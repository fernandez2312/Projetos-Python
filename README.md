# 🐍 Projetos em Python

Bem-vindo ao meu repositório de estudos! Este espaço foi criado para consolidar meu aprendizado na linguagem Python, reunindo exercícios de aulas, desafios de lógica e pequenos sistemas práticos.

## 📁 Estrutura do Repositório

O repositório está dividido em três projetos principais:

### 1. 📚 PythonProject-Aulas
* **Descrição:** Scripts, exercícios e anotações gerados durante as aulas teóricas.
* **Foco:** Sintaxe básica, estruturas condicionais (`if/else`), laços de repetição (`while/for`) e manipulação de variáveis.

### 2. ⚙️ PythonProject-Métodos-e-Funções
* **Descrição:** Códigos focados na modularização e reutilização de software.
* **Foco:** Criação de funções personalizadas, passagem de parâmetros, retorno de valores e escopo de variáveis.

### 3. 💼 PythonProject-Software-Gerenciamento-Bancario
* **Descrição:** Um pequeno sistema de simulação bancária.
* **Foco:** Lógica de negócios aplicada, controle de fluxo, operações matemáticas (depósito, saque, extrato) e persistência de dados simples.

---

## 🛠️ Tecnologias Utilizadas

* **Python 3.x**
* **Bibliotecas nativas** (ex: `math`, `os`, `sys`)

---

## 🚀 Como Executar os Projetos

1. Clone este repositório em sua máquina:
   ```bash
   git clone https://github.com
   ```
2. Acesse a pasta do projeto desejado:
   ```bash
   cd Nome-Da-Pasta
   ```
3. Execute o arquivo principal usando o terminal:
   ```bash
   python nome_do_arquivo.py
   ```
